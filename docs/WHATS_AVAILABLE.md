## üåê Data Sources

### Immediately Available (No API Keys)

#### 1. Yahoo Finance
- **What**: Stocks, indices, ETFs, crypto, commodities
- **Coverage**: Global markets
- **Historical**: Up to 20+ years
- **Cost**: Free
- **API Key**: Not required
- **Rate Limit**: Moderate

#### 2. CoinGecko
- **What**: 10,000+ cryptocurrencies
- **Coverage**: OHLC, volume, market cap
- **Historical**: Several years
- **Cost**: Free
- **API Key**: Not required
- **Rate Limit**: 50 calls/min

### With Free API Keys

#### 3. Alpha Vantage
- **What**: Stocks, forex, crypto, technical indicators
- **Coverage**: Professional-grade data
- **Get Key**: https://www.alphavantage.co/support/#api-key
- **Cost**: Free tier (5 calls/min, 500/day)
- **Features**: Intraday data, indicators, fundamentals

#### 4. FRED (Federal Reserve Economic Data)
- **What**: 800,000+ economic indicators
- **Get Key**: https://fred.stlouisfed.org/docs/api/api_key.html
- **Cost**: Free (generous limits)
- **Examples**:
  - `DFF` - Federal Funds Rate
  - `UNRATE` - Unemployment Rate
  - `CPIAUCSL` - Inflation (CPI)
  - `GDP` - Gross Domestic Product
  - `DGS10` - 10-Year Treasury Rate

#### 5. Polygon.io
- **What**: Stocks, options, forex, crypto
- **Get Key**: https://polygon.io/
- **Cost**: Free tier (5 calls/min, delayed data)
- **Features**: Real-time on paid tiers
