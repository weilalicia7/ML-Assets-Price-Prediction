## Search Capabilities

### Smart Search Aliases
Users can search by:
- **Ticker Symbol**: Direct entry (e.g., "AAPL", "600519.SS")
- **Company Name**: Full or partial (e.g., "Samsung", "Tencent")
- **Exchange**: Browse by market (e.g., "Shanghai", "Singapore")
- **Industry**: Commodity groups (e.g., "Gold", "Oil", "Semiconductors")
- **Common Names**: Brands (e.g., "Toyota", "Uniqlo", "Moutai")

### Examples:
- Search "**MOUTAI**" ‚Üí Finds Kweichow Moutai (600519.SS)
